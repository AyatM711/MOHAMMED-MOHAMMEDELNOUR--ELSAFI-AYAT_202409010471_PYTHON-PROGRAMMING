# Food Delivery System

## Description
This is a simple Python program that allows a user to enter a food order and prints a receipt with the total amount.

## Files
- `app.py` – Main program
- `customer.py` – Gets customer details
- `receipt.py` – Prints the receipt

## How to Run
Run the program using:

```bash
type : cd week_8
then 
python app.py
```
